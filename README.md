# Smart Request Routing System

## Overview
This project is a Smart Request Routing System developed using Django.  
It helps route customer service requests to the appropriate department automatically.

## Features
- User Authentication
- Request Management
- Staff Dashboard
- Smart Routing Logic
- Feedback System
- Status Tracking
- Admin Panel

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Installation

```bash
pip install -r requirements.txt
python manage.py runserver
```
